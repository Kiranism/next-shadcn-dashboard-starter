# Админ панель

Руководство по управлению проектами и настройками.

## Дашборд
`/dashboard`

Обзорная страница со статистикой:
- Количество проектов и пользователей.
- Графики активности.
- Список последних событий.

## Проекты
`/dashboard/projects`

Управление бонусными программами.

### Создание проекта
1. Нажмите **"Создать проект"**.
2. Укажите название и домен.
3. Выберите режим (с ботом или без).

### Настройки
`/dashboard/projects/[id]/settings`

- **Основные**: Название, домен, режим работы (EARN/SPEND).
- **Активация**: Правила входа для пользователей.
- **Webhook**: URL для интеграции и секретный ключ.

### Пользователи
`/dashboard/projects/[id]/users`

База клиентов проекта.
- **Поиск**: По email, телефону, имени.
- **Действия**: Начисление/списание бонусов, блокировка.
- **Экспорт**: Выгрузка базы в CSV.

### Аналитика
`/dashboard/projects/[id]/analytics`

Метрики эффективности:
- **Дашборд**: Оборот, заказы, средний чек, использование бонусов.
- **Когорты**: Анализ возвращаемости (Retention) по месяцам регистрации.
- **Финансы**: Ключевые метрики LTV, ARPU, ARPPU.
- **Рефералы**: Топ партнеров и выручка от приглашений.
- **Товары**: Рейтинг продаваемых позиций.

### Уровни лояльности
`/dashboard/projects/[id]/bonus-levels`

Настройка прогрессивной шкалы бонусов.
- Задайте пороги сумм покупок.
- Установите процент начисления для каждого уровня.

### Реферальная программа
`/dashboard/projects/[id]/referral`

Настройки виральности:
- Включение программы.
- Настройка процентов для уровней 1-3.
- Настройка cookie (срок жизни реферальной связи).
- Вкладка **«Планы %»** — персональные планы комиссий для блогеров/партнёров.
- Страница **«Иерархия»** (`/referral/hierarchy`) — дерево партнёров, статистика по команде, CSV-экспорт (при включённом B2B-режиме).

### Telegram бот
`/dashboard/projects/[id]/bot`

Подключение и настройка бота.
- Ввод токена (@BotFather).
- Тексты приветствий и автоответов.
- Меню и клавиатуры.

### Конструктор (Workflow)
`/dashboard/projects/[id]/workflow`

Визуальный редактор сценариев бота.
- Создание цепочек сообщений.
- Настройка триггеров и условий.
- Мониторинг выполнений.

### Рассылки
`/dashboard/projects/[id]/mailings`

Отправка уведомлений пользователям.
- Фильтрация по сегментам.
- Отложенная отправка.
- Статистика доставки.

### Интеграция
`/dashboard/projects/[id]/integrations`

Подключение внешних систем. Хаб интеграций показывает статус каждой и сколько настроено.

- **Webhook**: универсальный URL и секретный ключ для приёма событий.
- **Виджет**: JS-код для вставки на сайт (баланс, оплата бонусами).
- **Tilda**: виджет, бейджи бонусов на товарах, промокоды.
- **InSales**: webhook'и заказов/клиентов + виджет баланса ([инструкция](/insales-integration)).
- **МойСклад**: двусторонняя синхронизация бонусов онлайн ↔ офлайн ([инструкция](/moysklad-integration)).
- **MAX бот**: альтернативный мессенджер-бот.

### Логи
`/dashboard/projects/[id]/logs`

Журнал системных событий для отладки.
- Ошибки API.
- Действия пользователей.
- Ответы бота.

## Биллинг
`/dashboard/billing`

Управление подпиской.
- Текущий тариф и лимиты (число проектов и др.).
- История списаний.
- Смена плана.
- **Автопродление**: включение/отключение, управление привязанной картой.
- Оплата через ЮKassa.

> У администратора активен один тариф; лимит проектов определяется текущим планом.

## Профиль
`/dashboard/profile`

Настройки администратора.
- Смена пароля.
- Двухфакторная аутентификация.
- API ключи аккаунта.

### Как получить API Key
1. Перейдите в **Профиль**.
2. В блоке **API Keys** нажмите "Сгенерировать".
3. Скопируйте ключ (он показывается один раз). Используйте его в заголовке `X-API-Key`.
