# FAQ

Часто задаваемые вопросы.

## Общие

**Что такое Gupil?**
SaaS платформа для управления программами лояльности и Telegram ботами.

**Сколько стоит?**
Есть бесплатный тариф (до 100 пользователей). Платные тарифы расширяют лимиты. Подробнее в разделе Биллинг.

**Есть ли мобильное приложение?**
Управление происходит через веб-интерфейс (адаптивный). Клиенты используют Telegram.

## Настройка

**Как быстро запустить?**
1. Создайте проект.
2. Подключите Telegram бота.
3. Настройте Webhook интеграцию с сайтом.

**Какие данные нужны для интеграции?**
Минимум: Email или телефон клиента, сумма заказа, номер заказа.

**Нужно ли знать программирование?**
Нет, базовые функции работают через конструктор. API нужен только для сложной кастомной интеграции.

## Боты

**Что делать, если бот не отвечает?**
1. Проверьте статус в настройках (должен быть активен).
2. Проверьте токен.
3. Убедитесь, что активен хотя бы один сценарий (Workflow).

**Можно ли один бот на несколько проектов?**
Нет, 1 проект = 1 бот.

## Интеграции

**С какими платформами интегрируется Gupil?**
Tilda, InSales, МойСклад, а также любой сайт через универсальный виджет и webhook. Подробности — в разделах [Tilda](/tilda-integration), [InSales](/insales-integration), [МойСклад](/moysklad-integration), [Custom](/custom-integration).

**Можно ли синхронизировать офлайн-продажи?**
Да, через интеграцию с [МойСклад](/moysklad-integration) — бонусы синхронизируются между онлайн- и офлайн-каналами.

## Партнёры и рефералы

**Чем партнёрская программа (B2B) отличается от обычной реферальной?**
B2B-режим добавляет роли (тренер/менеджер/директор) и начисляет комиссию по цепочке вверх на несколько уровней. Включается отдельно для проекта. Подробнее — [Партнёрская программа (B2B)](/partner-program).

**Можно ли задать особый процент конкретному блогеру?**
Да, через персональные планы комиссий (вкладка «Планы %» в реферальной программе).

## Автоматизация

**Можно ли начислять бонусы ко дню рождения автоматически?**
Да, через триггеры по расписанию в [конструкторе сценариев](/workflow-constructor) — есть готовый шаблон «🎂 Бонусы ко дню рождения».

## Бонусы

**Как работает списание?**
При оплате заказа клиент может оплатить часть суммы бонусами (согласно настройкам уровня). Бонусы списываются через API.

**Сгорают ли бонусы?**
Да, если установлен срок действия в настройках проекта.

**Как работают уровни?**
Система автоматически повышает уровень клиента при достижении порога общей суммы покупок.
